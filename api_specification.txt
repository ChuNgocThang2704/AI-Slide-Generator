========================================================================
1. API SINH DỮ LIỆU JSON CẤU TRÚC SLIDE (KHÔNG SINH FILE PPTX)
========================================================================
API này dùng để lấy về cấu trúc slide chi tiết định dạng JSON (tiêu đề, các dòng nội dung, bố cục layout gợi ý, bảng biểu, biểu đồ và ảnh) để FE tự vẽ preview.

* Endpoint: /api/generate-slide-spec (hoặc alias: /generate-spec)
* Method: POST

CÁC THAM SỐ ĐẦU VÀO (REQUEST BODY - FORM FIELD):
- text (String, Không bắt buộc*): Nội dung văn bản cần làm slide.
- file (File, Không bắt buộc*): File tài liệu tải lên (hỗ trợ .docx, .pdf, .txt).
- content (String JSON, Không bắt buộc*): Dữ liệu cấu trúc slide thô nếu có sẵn.
- plan (String, Không bắt buộc): free (giới hạn 10 slide), pro (tối đa 30 slide - mặc định), ultra (tối đa 50 slide).
- slide_count (Integer, Không bắt buộc): Số slide yêu cầu (chỉ dùng cho gói pro/ultra).
- slide_theme (String, Không bắt buộc): modern / corporate / minimal.
- generate_images (String, Không bắt buộc): "true" hoặc "false" (mặc định là false).
- image_limit (Integer, Không bắt buộc): Giới hạn số lượng ảnh sinh ra.
- include_image_base64 (String, Không bắt buộc): Trả về base64 của ảnh trong JSON: "true" hoặc "false".

* Lưu ý: Phải truyền tối thiểu 1 trong 3 trường (text, file, hoặc content).

3. API KIỂM TRA TRẠNG THÁI TASK (POLLING API)
========================================================================
FE gọi lặp (Polling) API này mỗi 2 giây để kiểm tra tiến độ sinh slide.

* Endpoint: /api/status/{task_id}
* Method: GET
